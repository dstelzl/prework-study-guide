HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Accessibility is achieved using alt text and proper heading sizes
  • The main element contains the content on the page
  • The footer element contains author, copyright info, contact info, sitemap and navigation

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A . indicates "all" of that particular element will be affected bythe CSS

Git

  • git status checks what branch you are on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit saves your work
  • git pull origin main updates the branch to the main code

JavaScript

  • A variable is a named container that allows us to store datat in our code
  • Control flow is the oeder in which a computer executes code ina script
  • An array stores multiple data types in a single variable
  • A loop allows a single code block to be executed over and over
  • For loop statements contain 3 parts: Starting point, Condition and Iteration over each item
  • Functions to not automatically execute
